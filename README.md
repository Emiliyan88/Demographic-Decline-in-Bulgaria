# 📉 Demographic Decline in Bulgaria – Data Science Project

## 📌 Overview
This project analyzes the demographic decline in Bulgaria using real-world data.  
It focuses on population trends over time and applies machine learning techniques to forecast future population changes.

---

## 📊 Dataset
The dataset is based on World Bank demographic indicators and includes:
- Year
- Population data
- Additional socio-economic indicators (optional extensions)

Source: World Bank Data API

---

## 🎯 Objectives
- Analyze population trends in Bulgaria
- Visualize historical demographic changes
- Apply machine learning for forecasting
- Test statistical differences in population trends

---

## 🧠 Methods Used
- Data Cleaning & Preprocessing (Pandas)
- Data Visualization (Matplotlib, Seaborn)
- Linear Regression (Scikit-learn)
- Random Forest Regression
- Statistical Testing (T-test)

---

## 📈 Key Insights
- Bulgaria shows a consistent long-term population decline
- The trend is statistically significant before and after 2005
- Machine learning models confirm a downward trajectory

---

## 🤖 Machine Learning Models
- Linear Regression for trend forecasting
- Random Forest for comparison and error evaluation

---

## 📉 Results
- Clear negative population trend
- Forecast predicts continued decline
- Strong correlation between year and population decrease
